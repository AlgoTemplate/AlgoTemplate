??? note "题单食用提示"

    注 1：食用本题单时，可以先完成网站对应页面的讲解题，再完成对应的练习题。  

    注 2：题单中的题目没有提组别，后续标的推荐难度默认为 BCG 组的难度，而 A 组难度需要往上一个难度。如省二必会的题目，A 组推荐难度为省三必会。

    注 3：如果有不会的知识点，可以前往 [B站：算法竞赛入门](https://www.bilibili.com/video/BV1qucBz9EPv) 看视频讲解。

    当进入国赛后，国赛获奖难度为 C++ A 组最难（OIER 太多），C++ B 组其次（ACMER 太多），剩下的组别获奖难度差不多。

    题单难度为：国一稳固，国一必会，国二必会，国三必会。（没有特别说明，一般是默认为 C++ B 组国赛难度）
